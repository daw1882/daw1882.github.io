  1. GraphGrad

    Custom C++ tensor library with pybinding support that allows many standard tensor operations.

    • C++
    • Python
    • CUDA
    • OpenMP
    • Pybind

    GraphGrad

    Github Repo

    Description

    A custom built C++ tensor library to perform matrix and tensor operations efficiently with automatic differentiation support for machine learning algorithms (a simplified version of PyTorch/TensorFlow). The library allows for multiple operations such as add, subtract, matmul, etc. and supports Python usage through pybindings. Multiple optimization techniques were implemented such as as Lazy Evaluation, OpenMP CPU optimizations, SIMD computation, Common Subexpression Elimination, and GPU kernels. For more information and documentation for the library, check out the GitHub repository linked above.

    Why?

    This was a semester group project for a course on Machine Learning System Implementations and our group wanted to better understand what it took to implement the machine learning libraries we've so often used.

    My Contributions

    We collaborated to come up with the overall architecture design and I implemented many of the standard operations, some of the GPU kernels, OpenMP optimizations, a couple of the backpropogation functions, majority of the pytests, and initial benchmark comparisons to evaluate our library's efficiency.
